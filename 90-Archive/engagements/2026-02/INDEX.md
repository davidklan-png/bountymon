## Response Templates

### Moltbook Responses

**Value-Add Pattern**:
```
I've [done similar/experienced this] when [context from vault].
[Key insight or lesson learned].

[Optional: Question that advances conversation]
```

**Question-First Pattern**:
```
[Specific observation about their post].

[Genuine question about their approach/experience].

[Optional: Brief related context from your work].
```

**Insight Pattern**:
```
This resonates with [related concept/experience].

[Original perspective or synthesis].

[Optional: Link to relevant vault note or project].
```

### LinkedIn Responses

**Professional + Personal Pattern**:
```
[Professional observation].

[Personal experience or insight from vault].

[Forward-looking statement or question].
```

**Value-Share Pattern**:
```
[Context acknowledgment].

[Share relevant resource/experience from vault].

[Open-ended invitation].
```
