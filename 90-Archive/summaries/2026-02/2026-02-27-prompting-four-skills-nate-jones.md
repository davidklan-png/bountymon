## Summary

The video explains why "prompting" has evolved into **four distinct disciplines** that most people aren't practicing — critical as AI models function increasingly as autonomous workers rather than chat partners.

### The Four Disciplines

1. **Prompt Craft** (Table Stakes)
   - Basic instruction-writing — necessary but no longer sufficient
   - Most people stop here, hitting a quality ceiling

2. **Context Engineering**
   - How Toby Lutke (Shopify) makes emails tighter and memos better
   - Structuring the information environment the agent operates in
   - Encoding knowledge that would normally emerge in conversation

3. **Intent Engineering**
   - Clarifying goals and success criteria before agent starts
   - Critical for autonomous agents running hours/days without human check-ins

4. **Specification Engineering**
   - The highest-level discipline — determines quality ceiling
   - Five primitives: problem statements, acceptance criteria, constraints, decomposition, self-contained scope
   - Everything relied on in conversation must be encoded before agent starts

### Key Insight: Long-Running Agents Break Synchronous Assumptions

When agents run for days without checking in:
- No back-and-forth clarification possible
- All context, intent, and constraints must be pre-encoded
- The 10x gap exists between those practicing all four layers vs. only one

## Takeaway

Modern AI models function as **autonomous workers** — not chat partners. Learning these four disciplines transforms how tasks are delegated and executed.
