Simple, opinionated documentation platform. Book-like organization makes it easy to find and organize knowledge.

## Why it works

- **Book metaphor**: Books → Chapters → Pages (intuitive hierarchy)
- **WYSIWYG editor**: Write docs without learning Markdown
- **Search**: Full-text search across all content
- **LDAP/SAML**: Enterprise authentication built-in
- **API**: Automate content creation and updates

## What you lose

- Confluence's massive template library
- Some advanced collaboration features
- Enterprise-grade permission granularity
