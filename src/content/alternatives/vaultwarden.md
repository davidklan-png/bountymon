Lightweight Bitwarden-compatible password manager. Same clients, self-hosted backend, written in Rust.

## Why it works

- **Bitwarden-compatible**: Use official Bitwarden apps and browser extensions
- **Lightweight**: Runs on minimal hardware, < 100MB RAM
- **Full features**: TOTP, secure notes, shared collections, organizations
- **Easy migration**: Export from any password manager, import to Vaultwarden
- **Battle-tested**: Millions of downloads, active community

## What you lose

- Bitwarden's managed enterprise SSO
- Some advanced enterprise reporting features
